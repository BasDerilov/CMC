## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/BasDerilov/gcp_mc

# Access
$ cd gcp_mc

# Install dependencies
$ poetry install

# Run the project
$ poetry run gcp_mc_cli

# The server will be generated in the modules/palmsbet_mc/server folder
```
